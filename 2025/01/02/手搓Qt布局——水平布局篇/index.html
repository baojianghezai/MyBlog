手搓Qt布局——水平/竖直布局篇

OK啊，先立个小目标，一周至少两篇blog（新年小目标嘿嘿嘿）

第一步：先把ui文件以及相关引用全部删除

正如标题所说……，就是把.ui文件直接给删了…..

第二步：加上头文件，创建实例

一下操作都是在.cpp文件下进行

水平布局头文件：QHBoxLayout

创建实例： QHBoxLayout* pHLay = new QHBoxLayout(父指针); //父指针一般是this

第三步：添加控件

使用控件之前需要将控件包含进头文件

如：QLable,QLineEdit,QPushButton

每个控件都需要手动创建实例，并进行相关样式设计

关于QHBoxLayout类常用的方法

1.addLayout()

作用：在布局中添加布局

函数定义：void addLayout(QLayout *layout, int stretch = 0);

2.setSpacing()

作用：设置控件之间的间距，默认值为7

函数定义：void setSpacing(int spacing) override;

3.setContentsMargins()

作用：设置布局的边距

函数定义：void setContentsMargins(int left, int top, int right, int bottom);

“左上右下”原则

tips：qt6.x版本里面，去除了setMargin()这个函数，（别问我咋知道的，问豆包问出来的）

4.addStretch()

作用：添加水平弹簧(水平布局中)，竖直布局中是加竖直弹簧

函数定义：void addStretch(int stretch = 0);

竖直布局：QVBoxLayout

除了定义的时候需要用QVBoxLayout定义外，其它的和水平布局没别的不一样的
